Kalman filter学习笔记

前言

本文纯属笔记且未完成，巨坑，不具参考性。

应用

  • 数据融合
  • 目标追踪(SoRT)

前置知识点

  • 线性代数
  • 协方差，协方差矩阵
  • 正态分布

运动模型

运动模型可以分为

  • 一次运动模型
    • CV: 匀速模型(Constant Velocity)
    • CA: 匀加速模型(Constant Acceleration)
  • 二次运动模型
    • CTRV: 恒定转弯率和速度幅度模型(Constant Turn Rate and Velocity)
    • CTRA: 恒定转动率和加速度
    • CSAV: 恒定转向角和速度
    • CCA: 恒定曲率和加速度(Constant Curvature and Acceleration)

匀速模型例子

系统状态方程

x^k=Ax^k−1+wk−1\hat{x}_k = A\hat{x}_{k - 1} + w_{k - 1}x^k​=Ax^k−1​+wk−1​

  • x^k\hat{x}_kx^k​: 第k次预测值
  • wk−1w_{k-1}wk−1​: 过程误差协方差矩阵, 符合期望为0，协方差为Q的正态分布 P(w)∼N(0,Q)P(w) \sim \mathcal{N}(0, Q)P(w)∼N(0,Q)
  • A: 状态转移矩阵

观测/测量方程

zk=Hxk+vkz_{k} = Hx_{k} + v_kzk​=Hxk​+vk​

  • xkx_{k}xk​: 第k次观测值
  • vkv_{k}vk​: 测量误差协方差矩阵, 符合期望为0，协方差为R的正态分布 P(v)∼N(0,R)P(v) \sim \mathcal{N}(0, R)P(v)∼N(0,R)
  • H: 测量矩阵

计算步骤

Predication

x^kˉ=Ax^k−1\hat{x}_{\bar{k}} = A\hat{x}_{k - 1}x^kˉ​=Ax^k−1​

Pkˉ=APk−1AT+QP_{\bar{k}} = AP_{k - 1}A^T + QPkˉ​=APk−1​AT+Q

P: 状态协方差矩阵(state covariance matrix)

在使用时协方差矩阵P是一个迭代更新的量，每一轮预测和更新后，P都会更新一个新的值，因此初始化时可以根据估计协定，不用太苛求初始化的精准度，因为随着几轮迭代会越来越趋紧真实值。

Measurement update

Kk=PkˉHT(HPkˉ−1HT+R)−1K_k = P_{\bar{k}}H^T (HP_{\bar{k}-1}H^T + R)^{-1}Kk​=Pkˉ​HT(HPkˉ−1​HT+R)−1

x^k=x^kˉ+Kk(Zk−Hx^kˉ)\hat{x}_k = \hat{x}_{\bar{k}} + K_k(Z_k - H\hat{x}_{\bar{k}})x^k​=x^kˉ​+Kk​(Zk​−Hx^kˉ​)

Pk=(I−KkH)PkˉP_k = (I - K_kH)P_{\bar{k}}Pk​=(I−Kk​H)Pkˉ​

数据融合

X^k=X^k−1+Kk∗(Zk−X^k−1)\hat{X}_{k}=\hat{X}_{k-1} + K_k * (Z_k - \hat{X}_{k-1})X^k​=X^k−1​+Kk​∗(Zk​−X^k−1​)

  • X^\hat{X}X^: 当前估计值
  • X^k−1\hat{X}_{k-1}X^k−1​: 前一次估计值
  • ZkZ_kZk​: 当前测量值
  • KkK_kKk​: Kalman Gain,卡尔曼增益/卡尔曼系数

Kk=eESTk−1eESTk−1+eMEAkK_k = \frac{e_{EST_{k-1}}}{e_{EST_{k-1}} + e_{MEA_k}}Kk​=eESTk−1​​+eMEAk​​eESTk−1​​​

其中:

  • eEST{e}_{EST}eEST​: Error estimate, 估计误差
  • eMEA{e}_{MEA}eMEA​: Error measurement, 测量误差

eMEA{e}_{MEA}eMEA​一般由测量工具得出（传感器等）

当K=0时，eEST{e}_{EST}eEST​和X^\hat{X}X^可以为任何值，它将会在K=1时更新。

在K时刻

  • 当eEST≫eMEA{e}_{EST} \gg {e}_{MEA}eEST​≫eMEA​, 则 Kk→1K_k \to 1Kk​→1, Xk=ZkX_k = Z_kXk​=Zk​
  • 当eEST≪eMEA{e}_{EST} \ll {e}_{MEA}eEST​≪eMEA​, 则 Kk→0K_k \to 0Kk​→0, Xk=Xk−1X_k = X_{k-1}Xk​=Xk−1​

计算步骤

  1. K=0初始化eEST{e}_{EST}eEST​, X^\hat{X}X^, Zk{Z}_{k}Zk​, eMEA{e}_{MEA}eMEA​可忽略
  2. K = K + 1
  3. 根据eMEA{e}_{MEA}eMEA​和eESTk−1{e}_{EST_{k-1}}eESTk−1​​计算Kk{K}_{k}Kk​
  4. 从eMEA{e}_{MEA}eMEA​和Zk{Z}_{k}Zk​计算k^\hat{k}k^
  5. 更新eESTk{e}_{EST_{k}}eESTk​​
  6. 重复2
